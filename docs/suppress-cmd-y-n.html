    How to suppress "Terminate batch job (Y/N)" confirmation?

    Mar 30, 2021  - webdev Edit on GitHub!

    Install Clink оригинал или форк с поддержкой and change the “terminate_autoanswer” setting. The settings file should be here: C:\Users\<username>\AppData\Local\clink\settings.

    # name: Auto-answer terminate prompt 
# type: enum 
# Automatically answers cmd.exe's 'Terminate batch job (Y/N)?' prompts. 
0 = # disabled, 
1 = answer 'Y', 
2 = answer 'N'. 
terminate_autoanswer = 1

    or run clink set terminate_autoanswer 1 You can also see current value if you run this command without last argument. And you can even see the location of config file if you run just clink set

    У меня оно работает, но не на 100%. Позволяет нажать ctrl+c два раза для выключения.
