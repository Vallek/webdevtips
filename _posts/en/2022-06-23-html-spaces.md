```
+---------------------+-----------+
|      Character      | HTML Code |
+---------------------+-----------+
| Non-breaking space  | &nbsp;    |
| En space            | &ensp;    |
| Em space            | &emsp;    |
| Thin space          | &thinsp;  |
| Standard space      | &#20;     |
| New Line (Return)   | &#13;     |
| Tab Character       | &#09;     |
+---------------------+-----------+
```
