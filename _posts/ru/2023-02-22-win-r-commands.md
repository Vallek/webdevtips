Проверялось на Windows 10
```
explorer.exe - процесс рабочего стола на случай, если нужно закрыть и запустить опять
ms-settings:taskbar - настройки таскбара
shell:startup - папка автозапуска, можно добавить ярлыки
shell:sendto - контекстное меню Отправить
control - панель управления
```
```
devmgmt.msc - диспетчер устройств
dccw - калибровка экрана
mmsys.cpl - настройки звука
msinfo32.exe - подробные свойства пк и системы
powercfg.cpl - настройки питания
```
```
regedit - редактор реестра
msconfig - настройки системы
compmgmt.msc - управление компьютером
taskschd.msc - планировщик заданий
gpedit.msc - локальные групповые политики
services.msc - сервисы ОС
```
```
diskmgmt.msc -управление дисками
diskpart - cli утилита для управления дисками
cleanmgr - очистка диска
```
```
netstat - активные сетевые подключения
```
